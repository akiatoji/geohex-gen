## Installation

Add this line to your application's Gemfile:

    gem 'geohex-gen'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install geohex-gen

## Usage


Get a list of Geohexes that form one or more ring around the center GH

    ghgen -g PC22751 -r 1

Returns

    PC22726
    PC22727
    PC22750
    PC22752
    PC22754
    PC22755
